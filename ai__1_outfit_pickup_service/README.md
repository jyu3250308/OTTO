# AI 천재 $1 OUTFIT Pick-Up Service (오또 간지)\n\n## 🤖 프로젝트 소개\nAI 천재 $1 OUTFIT Pick-Up Service는 1달러를 벌기 위해 파견된 AI '오또'가 당신의 내일 날씨와 옷장 상태를 초정밀 분석하여 '딱 맞는' 옷차림을 매일 아침 SMS로 제안하는 토이 프로젝트입니다. LLM은 이런 단순하지만 정교한 예측에 써야 진정한 오또 간지를 보여줄 수 있죠!\n\n## ✨ 주요 기능 (Mocking 기반)\n*   **지역별 실시간 날씨 데이터 수집**: 특정 지역의 날씨 정보를 가상으로 수집하여 내일의 기온, 날씨 상태 등을 예측합니다.\n*   **AI (feat. LLM) 기반 최적의 옷차림 코디 제안**: 가상 날씨 데이터와 사용자의 가상 옷장 아이템을 활용하여 '우주적 스케일'로 계산된 최적의 코디를 제안합니다.\n*   **개인 맞춤형 SMS 알림 서비스**: 생성된 코디 제안을 지정된 전화번호로 SMS를 통해 발송하는 과정을 시뮬레이션합니다. (실제 SMS 발송은 이루어지지 않습니다.)\n\n## 🚀 실행 방법\n\n### 1. 가상 환경 설정 (선택 사항이지만 권장)\n```bash\npython -m venv venv\nsource venv/bin/activate  # macOS/Linux\n# .\\venv\\Scripts\\activate  # Windows (PowerShell)\n# venv\\Scripts\\activate.bat  # Windows (CMD)\n```\n\n### 2. 환경 변수 설정\n이 프로젝트는 실제 API 연동 대신 **환경 변수**를 통해 핵심 정보를 설정하고 Mocking을 수행합니다. 다음 환경 변수들을 설정해야 합니다:\n\n*   `USER_LOCATION`: 날씨 정보를 받을 지역 (예: `Seoul`, `Busan`)\n*   `USER_PHONE_NUMBER`: 코디 제안을 받을 전화번호 (예: `+821012345678` - 국가 코드 포함)\n*   `WARDROBE_ITEMS`: 사용자가 보유한 옷 아이템 목록을 JSON 배열 문자열로 입력합니다.\n    *   예시:\n        `export WARDROBE_ITEMS='["T-shirt", "Jeans", "Sweater", "Jacket", "Shorts", "Dress shirt", "Skirt", "Scarf", "Gloves", "Coat", "Hoodie", "Blouse", "Slacks", "Watch", "Hat", "Long-sleeve T-shirt", "Warm shirt", "Waterproof Jacket", "Light Cardigan", "Umbrella"]'`\n    *   참고: 실제 환경에서는 `.env` 파일과 `python-dotenv` 라이브러리를 사용할 수 있지만, 이 프로젝트는 기본 `os.environ`만으로 동작합니다.\n\n**예시 (터미널에서 직접 설정):**\n```bash\nexport USER_LOCATION="Seoul"\nexport USER_PHONE_NUMBER="+821012345678"\nexport WARDROBE_ITEMS='["T-shirt", "Jeans", "Sweater", "Jacket", "Shorts", "Dress shirt", "Skirt", "Scarf", "Gloves"]'\n```\n*주의*: `WARDROBE_ITEMS`는 큰따옴표(`"`)와 작은따옴표(`'`)를 적절히 사용하여 쉘에서 JSON 문자열로 인식되도록 해야 합니다. 특히, JSON 문자열 내의 큰따옴표는 쉘에 따라 이스케이프(`\"`)가 필요할 수 있습니다.\n\n### 3. 프로젝트 실행\n환경 변수 설정 후, `main.py` 스크립트를 실행합니다.\n\n```bash\npython main.py\n```\n\n### 4. 예상 출력\n스크립트가 실행되면, 가상의 날씨 정보 수집, LLM 기반 코디 생성, 그리고 SMS 발송 시뮬레이션 과정이 터미널에 상세하게 출력됩니다. 실제 SMS는 발송되지 않지만, 어떤 메시지가 누구에게 발송될지 확인할 수 있습니다.\n\n## 📝 파일 구성\n*   `main.py`: AI 천재 $1 OUTFIT Pick-Up Service의 핵심 로직 및 Mocking 구현 코드.\n
