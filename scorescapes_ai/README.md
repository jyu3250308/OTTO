## 🚀 무엇을 하는 봇인가요?

퍼블릭 도메인 명곡의 **실제 악보(음표 시퀀스)를 데이터로 읽어**, 음높이·음길이·멜로디 진행을
색·곡선·구도로 번역한 **포스터급 아트 PNG(1080×1350, 300DPI 인쇄 대응)** 를 만들어냅니다.

- API 키 불필요 · 인터넷 불필요 · 준비물은 `Pillow` 하나
- 실행할 때마다 색감이 달라지는 **세상에 하나뿐인 작품**
- 결과물은 `artworks/` 폴더에 실물 파일로 쌓입니다 (디지털 포스터·엽서·폰 배경으로 활용)

## 🛠️ 시작하기 (2단계 끝!)

```bash
pip install Pillow
python scorescapes_ai_bot.py
```

### 사용법

```bash
python scorescapes_ai_bot.py              # 오늘의 곡 랜덤 렌더링
python scorescapes_ai_bot.py "Arirang"    # 원하는 곡 지정 (부분 일치 OK)
python scorescapes_ai_bot.py list         # 수록곡 & 무드 목록
```

## 🎨 렌더링 엔진의 원리 (데이터 → 그림 번역 규칙)

1. **악보의 데이터화** — 각 곡은 `(MIDI 음높이, 박자 길이)` 시퀀스로 수록되어 있습니다.
2. **번역 규칙** — 음높이는 세로 위치·색을, 음길이는 크기·굵기를, 멜로디 진행은 화면의 흐름을 결정합니다.
3. **포스터 마감** — 2× 슈퍼샘플링(안티앨리어싱), 가우시안 글로우, 비네트, 필름 그레인, 미니멀 타이포까지 —
   인쇄해도 매끈한 마감입니다.

원리를 이해하면 `SCORES` 딕셔너리에 **좋아하는 곡의 음표를 직접 추가**해 나만의 무드 장면을 만들 수 있어요.

## 🔁 이렇게 활용해 보세요

- **매일 새 작품**: 윈도우 작업 스케줄러/cron에 등록하면 아침마다 새 아트가 갤러리에 쌓입니다.
- **선물/굿즈**: 300DPI라 그대로 인쇄소에 맡겨 엽서·포스터 제작이 가능합니다.
- **확장 아이디어**: MIDI 파일 파서를 붙이면 어떤 곡이든 자동으로 그림이 됩니다.
