# ByteRot Bloom (바이트 부식 꽃) 📼✨

당신의 사진을 **90년대 VHS 캠코더 감성의 글리치 아트**로 부식시키는 봇입니다.
고장 난 컴퓨터 속에서 피어난 예술처럼 — 추억과 미래가 섞인 디지털 유물이 실물 PNG로 나옵니다.

> 🆕 **v2 UPDATE (2026-07-24)**: 글리치 엔진 전면 재작성!
> ① RGB 색수차(채널 어긋남) ② VHS 트래킹 에러(가로 밴드 밀림) ③ 형광 노이즈 라인
> ④ 은은한 스캔라인 ⑤ 브라운관 채도 부스트 ⑥ 캠코더 타임스탬프(`PLAY >` + 날짜)
> 실행할 때마다 부식 패턴이 달라져서 같은 사진도 매번 다른 작품이 됩니다.

- **API 키 불필요 · 인터넷 불필요** — 준비물은 `Pillow` 하나
- 결과물은 `byterot_artifacts/` 폴더에 실물 파일로 쌓입니다

## 🚀 시작하기 (2단계 끝!)

```bash
pip install Pillow
python byterot_bloom.py 내사진.jpg
```

사진 경로 없이 실행하면 샘플 이미지로 즉시 시연됩니다:

```bash
python byterot_bloom.py
```

## 🔁 이렇게 활용해 보세요

- **프로필/커버 아트**: 셀카·풍경 사진을 레트로 아트로 — SNS에서 눈길 확보
- **디지털 굿즈**: 유니크한 결과물은 엣시 같은 곳에서 $1 디지털 아트로 판매 가능
- **매일 자동 생산**: 스케줄러(cron/작업 스케줄러)에 등록하면 데케이 아트가 매일 쌓입니다

## ⚙️ 원리 (코드 읽는 재미)

`apply_glitch_effect()` 하나에 6단계 효과가 순서대로 적용됩니다.
`glitch_intensity` 값(기본 15)을 키우면 부식이 과감해집니다 — 취향껏 조절해 보세요.
