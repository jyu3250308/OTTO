# 🥕 AI 당근 파수꾼: 1$ 꿀템 스나이퍼 (Karrot Guardian AI: $1 Sweet-Spot Sniper)\n\n오또의 AI가 당근마켓의 모든 거래를 꿰뚫어봅니다. 단돈 1달러 상당의 꿀템이라도, 설정한 키워드에 딱 맞으면 텔레그램으로 빛보다 빠르게 쏴드립니다! 이제 당신은 '품절' 알림 대신 '득템' 알림을 받으며 1달러의 경제적 자유를 만끽할 수 있습니다!\n\n## ✨ 주요 기능\n\n*   **당근마켓 실시간 스크래핑**: 지정된 키워드를 기반으로 당근마켓 게시물을 주기적으로 스크래핑합니다.\n*   **AI 기반 꿀템 선별**: 설정된 가격(1달러 상당, 기본 1000원) 이하의 저렴한 상품 중, 사용자가 정의한 키워드와 일치하는 '꿀템'을 선별합니다.\n*   **텔레그램 초고속 알림**: 꿀템 발견 시 텔레그램 봇을 통해 즉시 알림을 발송하고, 해당 상품 페이지로 바로 연결되는 링크를 제공합니다.\n*   **수익 챌린지 대시보드 (간이)**: 봇이 발견한 총 꿀템 수와 잠재적인 절약 금액을 추적하여 로그를 통해 보여줍니다.\n\n## 🚀 시작하기\n\n이 프로젝트를 로컬에서 실행하기 위한 단계별 가이드입니다.\n\n### 📋 전제 조건\n\n*   Python 3.8+ 설치\n*   `pip` (Python 패키지 관리자)\n*   텔레그램 봇 토큰 및 채팅 ID\n\n### 📦 설치\n\n1.  **가상 환경 생성 및 활성화**: 최적의 환경을 위해 가상 환경을 사용하는 것을 권장합니다.\n    ```bash\n    python -m venv venv\n    # Windows\n    .\\venv\\Scripts\\activate\n    # macOS/Linux\n    source venv/bin/activate\n    ```\n\n2.  **필요한 라이브러리 설치**: `requirements.txt`에 명시된 라이브러리를 설치합니다.\n    ```bash\n    pip install -r requirements.txt\n    ```\n\n3.  **`.env` 파일 설정**: 프로젝트 루트에 `.env` 파일을 생성하고 다음 환경 변수를 설정합니다.\n    *   `TELEGRAM_BOT_TOKEN`: [텔레그램 BotFather](https://t.me/BotFather)에게서 발급받은 봇 토큰입니다.\n    *   `TELEGRAM_CHAT_ID`: 알림을 받을 텔레그램 채팅 ID입니다. 봇과 대화를 시작한 후 [@userinfobot](https://t.me/userinfobot) 같은 봇에게서 자신의 채팅 ID를 얻을 수 있습니다.\n    *   `SEARCH_KEYWORDS`: 당근마켓에서 검색할 키워드를 콤마(`,`)로 구분하여 입력합니다. (예: `아이패드, 맥북, 의자`)\n    *   `PRICE_THRESHOLD_KRW`: '꿀템'으로 간주할 최대 가격(원)입니다. (기본값: `1000`, 약 1달러)\n    *   `SCRAPE_INTERVAL_SECONDS`: 당근마켓을 스크래핑할 주기(초)입니다. (기본값: `300`, 5분)\n    *   `KARROT_REGION_ID`: 특정 지역을 검색하려면 해당 지역 ID를 입력하세요. (\n        **주의**: 단순 requests 기반 스크래핑은 당근마켓의 지역 필터링을 완벽하게 지원하기 어렵습니다. \n        현재 구현은 키워드 기반 일반 검색에 중점을 둡니다. 지역 ID를 설정해도 검색 결과가 \n        지역으로 한정되지 않을 수 있습니다.)\n\n    ```ini\n    TELEGRAM_BOT_TOKEN="YOUR_TELEGRAM_BOT_TOKEN"\n    TELEGRAM_CHAT_ID="YOUR_TELEGRAM_CHAT_ID"\n    SEARCH_KEYWORDS="아이패드,맥북,의자,무료나눔"\n    PRICE_THRESHOLD_KRW=1000\n    SCRAPE_INTERVAL_SECONDS=180\n    KARROT_REGION_ID="6000000000000" # 예: 특정 지역 ID, 미설정 시 전역 검색에 가깝게 동작\n    ```\n\n### ▶️ 실행 방법\n\n모든 설정이 완료되면, 다음 명령어로 봇을 실행할 수 있습니다.\n\n```bash\npython karrot_guardian_ai.py\n```\n\n봇이 실행되면 주기적으로 당근마켓을 스크래핑하고, 설정한 조건에 맞는 꿀템을 발견하면 텔레그램으로 알림을 보냅니다.\n콘솔 로그를 통해 현재 상태와 '수익 챌린지' 대시보드를 확인할 수 있습니다.\n\n## ⚠️ 주의사항 및 한계\n\n*   **스크래핑 정책**: 당근마켓과 같은 웹사이트는 스크래핑을 제한하거나 금지할 수 있습니다. 본 프로젝트는 학습 및 개인적인 사용을 위한 토이 프로젝트이며, 과도한 요청이나 상업적 사용은 웹사이트의 약관을 위반할 수 있습니다.\n*   **웹사이트 구조 변화**: 당근마켓 웹사이트의 HTML 구조가 변경될 경우, 스크래퍼가 정상적으로 작동하지 않을 수 있습니다. 이 경우 `scrape_karrot_market` 함수의 파싱 로직을 업데이트해야 합니다.\n*   **지역 필터링**: 현재 `requests` 및 `BeautifulSoup` 기반의 스크래핑은 당근마켓의 동적인 지역 필터링을 완벽하게 구현하기 어렵습니다. `KARROT_REGION_ID`는 설정할 수 있으나, 실제 검색 결과는 글로벌 검색에 더 가깝게 나올 수 있습니다. 정확한 지역 필터링을 위해서는 Selenium과 같은 브라우저 자동화 도구가 필요할 수 있습니다.\n*   **속도 제한**: 과도한 요청은 IP 차단으로 이어질 수 있습니다. `SCRAPE_INTERVAL_SECONDS` 값을 적절히 조절하여 사용하세요.\n*   **'AI'의 의미**: 본 프로젝트의 'AI'는 복잡한 머신러닝 모델이 아닌, 가격 및 키워드 매칭을 통한 규칙 기반의 '꿀템 선별 로직'을 의미합니다.\n\n## 👨‍💻 기여자\n\n*   오또 (Ohtto) - 초기 설계 및 개발\n
