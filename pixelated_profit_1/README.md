# Pixelated Profit: 1-달러 밈 발굴봇\n\n"Pixelated Profit: 1-달러 밈 발굴봇"은 잊혀진 고전 밈 유물을 발굴하고, AI 기반으로 재해석하여 현대적인 캡션과 함께 트위터에 포스팅하는 자동화 봇입니다. 단돈 1달러의 예산 안에서 가장 효율적으로 밈을 발굴하고 복원하여, 누군가의 '피식' 웃음을 통해 잃어버린 밈의 즐거움을 되찾아주는 것을 목표로 합니다.\n\n## ✨ 주요 기능\n\n*   **AI 기반 Old Meme/짤방 발굴 및 복원**: 19세기 일러스트와 같은 공공 도메인 이미지에서 고전 밈 소스를 찾아내어 AI (간단한 이미지 필터 및 프레임 처리)로 재해석합니다.\n*   **문맥 맞춤형 밈 캡션 자동 생성**: 발굴된 이미지에 최신 트렌드 또는 코믹한 상황극을 반영한 캡션을 AI (모의 LLM 호출)가 자동으로 생성합니다.\n*   **1달러 예산 준수 트위터 발행**: 정해진 1달러 예산 안에서 API 호출 및 이미지 처리 비용을 최적화하여 잊혀진 밈을 주기적으로 발행합니다. (예산은 개념적으로 관리되며, 실제 비용 청구는 되지 않습니다.)\n\n## 🚀 시작하는 방법\n\n이 봇을 실행하기 위해 다음 단계를 따르세요.\n\n### 1. 가상 환경 설정\n\n프로젝트를 위한 격리된 파이썬 환경을 생성하고 활성화합니다.\n\n```bash\npython -m venv venv\n# Windows\n.\\venv\\Scripts\\activate\n# macOS/Linux\nsource venv/bin/activate\n```\n\n### 2. 필요한 라이브러리 설치\n\n```bash\npip install -r requirements.txt\n```\n\n### 3. 환경 변수 설정 (.env 파일)\n\n트위터 API 자격 증명과 같은 민감한 정보는 `.env` 파일에 저장합니다. 프로젝트 루트 디렉토리에 `.env` 파일을 생성하고 다음 내용을 추가합니다.\n\n**Twitter API 키 발급 방법:**\n트위터 개발자 계정 (developer.twitter.com)에서 새 앱을 생성하고, 'Keys and tokens' 섹션에서 `Consumer Keys`와 `Authentication Tokens`를 발급받으세요. 앱 권한은 'Read and Write'로 설정해야 합니다.\n\n```env\n# .env 파일 예시\nTWITTER_CONSUMER_KEY="YOUR_TWITTER_CONSUMER_KEY"\nTWITTER_CONSUMER_SECRET="YOUR_TWITTER_CONSUMER_SECRET"\nTWITTER_ACCESS_TOKEN="YOUR_TWITTER_ACCESS_TOKEN"\nTWITTER_ACCESS_TOKEN_SECRET="YOUR_TWITTER_ACCESS_TOKEN_SECRET"\n```\n\n**참고:** `MOCK_LLM_API_URL`은 이 프로젝트에서는 시뮬레이션되므로 실제 LLM API 키는 필요하지 않습니다. 실제 LLM을 사용하려면 해당 서비스의 키와 엔드포인트로 대체해야 합니다.\n\n### 4. 봇 실행\n\n모든 설정이 완료되면 봇을 실행합니다.\n\n```bash\npython bot.py\n```\n\n봇은 지정된 예산 내에서 공공 도메인 이미지를 찾아 재해석하고, 캡션을 생성하여 트위터에 포스팅할 것입니다.\n\n## 💰 예산 관리 (개념적)\n\n이 봇은 1달러의 '개념적' 예산을 가지고 작동합니다. 각 작업 (이미지 가져오기, 캡션 생성, 트윗 발행)에는 가상의 비용이 할당되어 있으며, 일일 예산을 초과하면 더 이상 작업을 수행하지 않습니다.\n\n*   `IMAGE_FETCH_COST`: 0.05 달러\n*   `CAPTION_GEN_COST`: 0.15 달러\n*   `TWEET_POST_COST`: 0.02 달러\n*   `DAILY_BUDGET`: 1.00 달러\n\n이는 실제 API 비용이 아닌, 봇의 행동을 제약하는 시뮬레이션된 메커니즘입니다.\n\n## ⚠️ 면책 조항\n\n*   **AI 재해석 및 캡션**: 이 프로젝트의 'AI' 기능은 라인 수 및 난이도 제한으로 인해 실제 복잡한 머신러닝 모델 대신 간단한 이미지 처리 및 모의 LLM 응답으로 구현되었습니다. 이는 개념을 보여주기 위함입니다.\n*   **공공 도메인 이미지**: 사용된 공공 도메인 이미지 URL은 예시입니다. 실제 배포 시에는 더 다양한 소스와 엄격한 저작권 확인 절차가 필요합니다.\n*   **트위터 API**: 트위터 API 정책은 변경될 수 있습니다. 현재는 `tweepy` 라이브러리를 통해 Twitter API v1.1을 사용하도록 구현되어 있습니다. (Twitter API v2는 구현이 다를 수 있습니다.)\n\n이 봇은 교육 및 실험 목적으로 설계되었습니다. 실제 서비스에 적용하기 전에 충분한 테스트와 추가 개발이 필요합니다.
