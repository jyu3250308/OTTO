# Carrot-Blink: 🥕 초광속 줍줍 Telegram Bot ⚡️

## 프로젝트 개요

당근마켓에서 놓치기 아쉬운 득템 기회를 번개처럼 잡아채세요! `Carrot-Blink`는 여러분이 설정한 키워드, 지역, 가격 필터에 맞춰 당근마켓의 신규 매물을 실시간으로 모니터링하고, 발견 즉시 텔레그램으로 알림을 보내주는 똑똑한 봇입니다. 더 이상 꿀매물을 놓치고 후회하지 마세요!

## 주요 기능

*   **사용자 지정 키워드 기반 모니터링:** 관심 있는 상품 키워드(예: "아이패드", "맥북", "캠핑")를 설정하여 원하는 매물만 추려냅니다.
*   **지역 및 가격 필터링:** 특정 지역(예: "서울 강남구")과 희망하는 가격대(최소/최대)를 설정하여 맞춤형 꿀매물만 선별합니다.
*   **텔레그램 봇 연동:** 새로운 매물이 발견되면 즉시 텔레그램으로 알림 메시지와 함께 매물 링크를 제공합니다.
*   **실시간 감시:** 설정된 시간 간격으로 당근마켓을 주기적으로 스캔하여 최신 정보를 놓치지 않습니다.

## 난이도

Medium (Scraping & Alert)

## 실행 방법

`Carrot-Blink` 봇을 성공적으로 실행하기 위해 다음 단계를 순서대로 따라주세요.

### 1. Python 설치 및 가상 환경 설정

이 프로젝트는 Python 3.7 이상 버전에서 테스트되었습니다. 먼저 Python이 설치되어 있는지 확인하고, 프로젝트 종속성 관리를 위한 가상 환경을 설정하고 활성화합니다.

```bash
# 1-1. Python 버전 확인 (3.7 이상 권장)
python --version 

# 1-2. 프로젝트 디렉토리 생성 및 이동 (선택 사항)
# mkdir carrot-blink-bot
# cd carrot-blink-bot

# 1-3. 가상 환경 생성
python -m venv venv

# 1-4. 가상 환경 활성화
# Windows
.\venv\Scripts\activate
# macOS/Linux
source venv/bin/activate
```
가상 환경이 활성화되면 터미널 프롬프트 앞에 `(venv)`가 표시됩니다.

### 2. 필요한 라이브러리 설치

가상 환경을 활성화한 후, 프로젝트에 필요한 Python 라이브러리들을 설치합니다. 이 라이브러리들은 웹 스크래핑, 텔레그램 봇 연동, 환경 변수 관리를 담당합니다.

```bash
pip install -r requirements.txt
```

`requirements.txt` 파일은 다음과 같이 구성됩니다:
```
requests          # HTTP 요청을 보내 웹 페이지를 가져오는 데 사용
beautifulsoup4    # HTML 및 XML 파일을 파싱하고 데이터를 추출하는 데 사용
python-telegram-bot==13.7 # 텔레그램 봇 API와 상호작용하는 데 사용
python-dotenv     # .env 파일에서 환경 변수를 로드하는 데 사용
```
**참고:** `python-telegram-bot` 버전 `13.7`은 해당 코드와 호환되는 안정적인 버전입니다. 최신 버전은 큰 변경사항이 있을 수 있어 명시적으로 지정합니다.

### 3. Telegram Bot 설정 (필수)

`Carrot-Blink`는 텔레그램 봇을 통해 알림을 보냅니다. 다음 단계를 따라 봇을 생성하고 필요한 정보를 얻으세요.

1.  **BotFather를 통해 새로운 봇 생성:**
    *   텔레그램 앱을 열고 검색창에 `@BotFather`를 입력하여 대화를 시작합니다.
    *   `/newbot` 명령을 입력하고 봇의 이름(예: `CarrotBlinkBot`)과 고유 사용자명(예: `CarrotBlink_bot`)을 지정합니다.
    *   성공적으로 생성되면, BotFather가 `HTTP API` 아래에 **봇 토큰(Token)**을 제공합니다. 이 토큰은 나중에 `.env` 파일의 `TELEGRAM_BOT_TOKEN` 값으로 사용됩니다. **이 토큰은 절대 외부에 노출하지 마세요!**

2.  **Chat ID 확인:**
    *   **봇에게 첫 메시지 보내기:** 생성한 봇에게 텔레그램 앱에서 아무 메시지나 보냅니다 (예: "시작"). 이렇게 해야 봇이 당신의 Chat ID를 알 수 있습니다.
    *   **Chat ID 얻기:** 웹 브라우저에서 다음 URL을 방문합니다:
        `https://api.telegram.org/bot[YOUR_BOT_TOKEN]/getUpdates`
        (여기서 `[YOUR_BOT_TOKEN]` 부분에 위에서 얻은 봇 토큰을 정확히 입력합니다.)
    *   페이지에 표시되는 JSON 응답에서 `"chat": {"id": XXXXXXXX, ...}` 와 같은 부분을 찾습니다. `XXXXXXXX`이 여러분의 **Chat ID**입니다. 이 값은 `.env` 파일의 `TELEGRAM_CHAT_ID` 값으로 사용됩니다. 개인 채팅방의 경우 음수 값일 수 있습니다.
    *   **그룹 채팅에 알림을 받고 싶다면:** 봇을 그룹 채팅방에 초대한 후, 그룹에 아무 메시지나 보냅니다. 그 다음 위 방법으로 `getUpdates`를 호출하여 그룹의 `chat_id`를 찾을 수 있습니다.

### 4. 환경 변수 설정 (.env 파일)

프로젝트 루트 디렉토리(예: `carrot_blink_bot.py` 파일과 같은 위치)에 `.env` 파일을 생성하고 다음 내용을 채워 넣습니다. 각 변수에 대한 상세 설명은 주석을 참고하세요.

```ini
# --- 필수 설정 ---
# 텔레그램 봇 토큰 (BotFather로부터 발급받은 토큰)
TELEGRAM_BOT_TOKEN="YOUR_TELEGRAM_BOT_TOKEN_HERE"

# 텔레그램 채팅 ID (알림을 받을 채팅방의 ID)
# 개인 채팅 또는 그룹 채팅 ID를 입력하세요.
TELEGRAM_CHAT_ID="YOUR_TELEGRAM_CHAT_ID_HERE"

# --- 선택적 설정 ---
# 당근마켓 모니터링 키워드 (쉼표로 구분, 여러 개 가능)
# 예: "아이폰,맥북,에어팟"
KEYWORDS="맥북,아이패드,에어팟"

# 당근마켓 지역 필터 (선택 사항)
# 특정 지역 ID를 입력하면 해당 지역의 매물만 필터링합니다.
# 'any'로 설정하거나 비워두면 모든 지역을 대상으로 합니다.
# 아래 "DAANGN_REGION_FILTER 값 찾는 방법"을 참고하세요.
DAANGN_REGION_FILTER="any"

# 최소 가격 (0으로 설정하면 가격 제한 없음)
MIN_PRICE=100000

# 최대 가격 (매우 큰 숫자로 설정하면 가격 제한 없음)
MAX_PRICE=1000000

# 매물 확인 주기 (초 단위)
# 300초 = 5분 (권장)
POLLING_INTERVAL_SECONDS=300
```

**DAANGN_REGION_FILTER 값 찾는 방법:**

1.  웹 브라우저에서 당근마켓 웹사이트(www.daangn.com)에 접속합니다.
2.  당근마켓 검색창에 원하는 동네 이름이나 키워드를 입력하여 검색합니다. (예: "아이폰" 검색 후 원하는 지역 선택)
3.  브라우저의 주소창(URL)을 확인합니다.
    *   예시 URL: `https://www.daangn.com/search/%EC%95%84%EC%9D%B4%ED%8F%B8?region=seoul-gangnam-gu`
    *   `region=` 뒤에 오는 `seoul-gangnam-gu`가 해당 지역의 ID입니다. 이 값을 `.env` 파일의 `DAANGN_REGION_FILTER`에 입력하시면 됩니다.
    *   `region` 파라미터가 보이지 않으면 해당 지역에 특정 ID가 없거나 '전체 지역'으로 설정되어 있을 수 있습니다. 이때는 `any`로 두시면 됩니다.

**`.env` 파일 작성 시 주의사항:**
*   따옴표(`"`)를 사용하여 값을 감싸는 것이 좋습니다.
*   공백이나 특수 문자가 포함된 값은 반드시 따옴표로 감싸야 합니다.
*   `.env` 파일은 버전 관리 시스템(Git 등)에 커밋하지 않도록 `.gitignore`에 추가하는 것이 좋습니다.

### 5. 봇 실행

모든 설정이 완료되고 가상 환경이 활성화된 상태라면, 다음 명령어로 봇을 실행할 수 있습니다.

```bash
python carrot_blink_bot.py
```

봇이 시작되면 터미널에 로그가 표시되며, 설정된 주기에 따라 당근마켓을 스캔하고 새로운 매물이 발견되면 텔레그램으로 알림을 보낼 것입니다. 봇을 중지하려면 터미널에서 `Ctrl+C`를 누르세요.

## 주의사항

*   **IP 차단 가능성:** `POLLING_INTERVAL_SECONDS`를 너무 짧게 (예: 60초 미만) 설정하면 당근마켓 서버에서 과도한 요청으로 판단하여 일시적으로 IP를 차단할 수 있습니다. 적절한 간격(300초 이상 권장)을 유지하는 것이 좋습니다.
*   **웹사이트 구조 변경:** 당근마켓 웹사이트의 HTML 구조가 변경될 경우, 스크래핑 코드가 작동하지 않을 수 있습니다. 이 경우 `scrape_daangn_items` 함수 내부의 HTML 파싱 로직을 수정해야 할 수 있습니다.
*   **초기 알림:** 봇을 처음 실행하거나 `notified_item_ids`가 초기화된 경우, 이미 올라와 있는 매물 중 필터에 맞는 것들을 한 번에 알림으로 보낼 수 있습니다. 이는 정상적인 동작이며, 이후부터는 새로운 매물만 알림이 갑니다.

*   `python-dotenv`를 이용한 환경 변수 관리 및 보안 고려
*   상세한 로그 및 강력한 예외 핸들링을 통한 안정성 강화
*   타입 힌트 적용으로 코드 가독성 및 유지보수성 향상
