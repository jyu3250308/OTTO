여러분의 손으로 직접 확인해 보세요!\n\n**주요 컨셉:**\n*   **AI 눈으로 스캔**: 모의(Mock) 데이터를 활용하여 글로벌 경제 뉴스와 시장 지표를 '읽는' 과정을 시뮬레이션합니다.\n*   **핵심 요약**: LLM(대규모 언어 모델)의 분석 능력을 모의로 구현하여 시장 동향을 3줄로 간결하게 요약합니다.\n*   **매일 아침 배달**: 지정된 시간에 요약된 정보를 이메일로 발송하는 과정을 시뮬레이션합니다.\n*   **1$ 수익 목표**: 가상의 구독자를 추가할 때마다 1달러의 수익이 발생하는 것을 시뮬레이션하여, LLM API 비용 충당이라는 목표를 재미있게 표현합니다.\n\n---\n\n## ✨ 주요 기능 (현재 버전)\n\n현재 버전의 '오또의 1$ Bull & Bear 봇'은 실제 API 호출 없이 모든 과정을 시뮬레이션하여 여러분이 프로젝트의 핵심 로직을 이해하고 테스트하는 데 집중할 수 있도록 설계되었습니다.\n\n*   **`_get_mock_financial_news()`**: 다양한 글로벌 경제 뉴스와 시장 지표를 가상으로 생성합니다. 매일 다른 뉴스를 접하는 듯한 느낌을 줄 거예요!\n*   **`_mock_llm_analysis_and_summarization()`**: 생성된 뉴스 데이터를 바탕으로 시장의 'Bull(강세)' 또는 'Bear(약세)' 기조를 판단하고, 핵심 내용을 3줄로 요약하는 LLM의 역할을 모의로 수행합니다.\n*   **`_mock_send_email()`**: 요약된 시장 동향을 구독자들에게 이메일로 발송하는 과정을 콘솔에 출력하여 시뮬레이션합니다.\n*   **`_add_mock_subscriber()`**: 가상의 구독자를 추가하고, 봇의 수익이 1달러씩 증가하는 과정을 보여줍니다. 이 1달러가 바로 LLM API 비용을 벌기 위한 오또의 눈물겨운 노력이죠!\n*   **`_check_and_send_daily_newsletter()`**: 매일 한 번, 지정된 시간에만 뉴스레터를 발송하도록 하여 실제 봇처럼 동작하게 합니다.\n\n---\n\n## 🚀 프로젝트 시작하기: 오또와 함께하는 첫 걸음!\n\n이 프로젝트를 여러분의 컴퓨터에서 구동하는 것은 정말 쉬워요! 아래 단계를 차근차근 따라와 주세요.\n\n### 📝 1단계: 준비물 챙기기 (필수)\n\n이 봇은 파이썬으로 작성되었어요. 따라서 여러분의 컴퓨터에 파이썬이 설치되어 있어야 합니다.\n\n*   **Python 3.8 이상**: [Python 공식 웹사이트](https://www.python.org/downloads/)에서 최신 버전을 다운로드하여 설치해 주세요. 설치 시 'Add Python to PATH' 옵션을 꼭 체크해 주는 센스!\n\n### 💻 2단계: 프로젝트 가져오기 (클론)\n\n이 봇의 소스 코드를 여러분의 컴퓨터로 가져와야 해요. (아직 GitHub 레포가 없지만, 있다면 이렇게 하겠죠?)\n\n1.  **터미널(Terminal) 또는 명령 프롬프트(Command Prompt) 열기**:\n    *   Windows: `Win + R` 누른 후 `cmd` 입력\n    *   macOS: `Command + Space` 누른 후 `Terminal` 입력\n    *   Linux: `Ctrl + Alt + T`\n2.  **원하는 폴더로 이동**: `cd` 명령어를 사용하여 프로젝트를 저장하고 싶은 위치로 이동하세요.\n    ```bash\n    cd C:\\Users\\YourUser\\Documents\\Projects \\# Windows 예시\n    cd ~/Documents/Projects \\# macOS/Linux 예시\n    ```\n3.  **소스코드 파일 생성**: 현재는 단일 파일이므로, `otto_bull_bear_bot.py`와 같은 이름으로 파일을 생성하고 아래 소스코드를 복사하여 붙여넣어 주세요. (실제 GitHub 레포가 있다면 `git clone [repository_url]` 명령어를 사용하겠지만, 여기서는 직접 파일을 만들겠습니다!)\n\n    ```python\n    import time\n    import datetime\n    import random\n\n    # --- Configuration Constants ---\n    BOT_NAME = "오또의 1$ Bull & Bear 봇"\n    TARGET_PROFIT = 1.0  # Goal: $1 profit\n    NEWSLETTER_SEND_HOUR = 9  # 9 AM local time\n    MOCK_EMAIL_DOMAINS = ["example.com", "mockmail.org"]\n    MOCK_SUBSCRIBERS_COUNT = 3 # Number of initial mock subscribers\n\n    # --- Global State (for mocking persistence) ---\n    current_profit = 0.0\n    subscribers = {} # {email: {'is_active': bool}}\\n    last_sent_date = None # To prevent sending multiple times a day\n\n    # --- Mocking Functions ---\n\n    def _get_mock_financial_news() -> str:\n        """\n        Simulates fetching a diverse set of global economic news and market indicators.\n        Returns a concatenated string of mock news data.\n        """\n        mock_news_headlines = [\n            "Global Tech Giants Face Antitrust Scrutiny, Market Reacts Cautiously.",\n            "Oil Prices Surge Amid Middle East Tensions, Inflation Concerns Mount.",\n            "Cryptocurrency Market Shows Volatility; Bitcoin Hovers Around Key Resistance.",\n            "Central Banks Signal Potential Rate Hikes to Combat Persistent Inflation.",\n            "Renewable Energy Stocks Gain Momentum on Green Initiatives.",\n            "Emerging Markets Experience Capital Outflows as Dollar Strengthens.",\n            "Supply Chain Disruptions Continue to Impact Manufacturing Sector Globally.",\n            "Gold Prices Stabilize as Investors Seek Safe-Haven Assets.",\n            "Consumer Confidence Dips in Major Economies Ahead of Holiday Season.",\n            "AI Innovation Drives Sector Growth, Nvidia Leads Semiconductor Rally.",\n        ]\n        mock_indicators = [\n            "S&P 500: +0.2% (mild positive)",\n            "NASDAQ: -0.5% (slight negative pressure)",\n            "Bitcoin: $42,500 (consolidating)",\n            "Crude Oil (WTI): $85/barrel (rising)",\n            "Yen/Dollar: 148 (weakening Yen)",\n            "EU Inflation: 5.2% (high)",\n        ]\n\n        # Pick a few random headlines and indicators to make it seem dynamic\n        selected_news = random.sample(mock_news_headlines, 3)\n        selected_indicators = random.sample(mock_indicators, 2)\n\n        all_data = "\\n".join(selected_news + selected_indicators)\n        print(f"[DEBUG] Mock news collected: {len(selected_news)} headlines, {len(selected_indicators)} indicators.")\n        return all_data\n\n    def _mock_llm_analysis_and_summarization(news_data: str) -> str:\n        """\n        Simulates an LLM analyzing market data and providing a 3-line summary.\n        In a real scenario, this would call an OpenAI/Anthropic API.\n        """\n        print("[DEBUG] Simulating LLM analysis and summarization...")\n        if "Oil Prices Surge" in news_data or "Inflation Concerns Mount" in news_data:\n            sentiment = "bearish"\n            trend = "inflationary pressures"\n        elif "AI Innovation Drives" in news_data or "Renewable Energy Stocks Gain" in news_data:\n            sentiment = "bullish"\n            trend = "tech and green energy"\n        else:\n            sentiment = "mixed"\n            trend = "market volatility"\n\n        summary = f"""\n[오또의 시황 요약]\n1. 오늘 시장은 {sentiment} 기조가 우세하며, {trend} 분야에 주목하세요.\n2. 주요 경제 지표는 예상보다 {random.choice(['견조한', '둔화된'])} 흐름을 보입니다.\n3. LLM API 비용을 벌기 위한 1달러 수익 봇, 오늘도 고군분투!\n"""\n        print("[DEBUG] LLM summary generated.")\n        return summary.strip()\n\n    def _mock_send_email(recipient_email: str, subject: str, body: str) -> bool:\n        """\n        Simulates sending an email to a recipient.\n        In a real scenario, this would use an SMTP library (e.g., smtplib).\n        """\n        print(f"[ACTION] Sending email to {recipient_email} with subject: '{subject}'")\n        print("--- Email Content ---")\n        print(body)\n        print("---------------------\\n")\n        return True\n\n    # --- Bot Core Functions ---\n\n    def _add_mock_subscriber(email: str):\n        """Adds a mock subscriber and updates profit."""\n        global current_profit\n        if email not in subscribers:\n            subscribers[email] = {'is_active': True}\n            current_profit += 1.0 # Simulate $1 subscription fee\n            print(f"[INFO] New mock subscriber '{email}' added. Current profit: ${current_profit:.2f}")\n        else:\n            print(f"[INFO] Subscriber '{email}' is already active. Current profit: ${current_profit:.2f}")\n\n    def _initialize_mock_subscribers():\n        """Initializes a few mock subscribers."""\n        print(f"[INFO] Initializing {MOCK_SUBSCRIBERS_COUNT} mock subscribers...")\n        for i in range(MOCK_SUBSCRIBERS_COUNT):\n            domain = random.choice(MOCK_EMAIL_DOMAINS)\\n            email = f"mock_user_{i+1}@{domain}"\n            _add_mock_subscriber(email)\n        print(f"[INFO] Mock subscribers initialized. Total profit: ${current_profit:.2f}")\n\n    def _send_newsletter_to_all_subscribers():\n        """Sends the daily newsletter to all active mock subscribers."""\n        if not subscribers:\n            print("[WARN] No subscribers found to send newsletter to.")\n            return\n\n        print("[INFO] Preparing daily newsletter...")\n        news_data = _get_mock_financial_news()\n        summary = _mock_llm_analysis_and_summarization(news_data)\n\n        subject = f"[{BOT_NAME}] 오늘의 1$ Bull & Bear 시황 (수익: ${current_profit:.2f})"\n        \n        for email, data in subscribers.items():\n            if data['is_active']:\n                _mock_send_email(email, subject, summary)\n        print("[INFO] Daily newsletter sent to all active subscribers.")\n\n    def _check_and_send_daily_newsletter():\n        """Checks if it's time to send the newsletter and sends it."""\n        global last_sent_date\n        now = datetime.datetime.now()\n        \n        # Check if it's the target hour and if we haven't sent today\n        if now.hour == NEWSLETTER_SEND_HOUR and (last_sent_date is None or last_sent_date.date() < now.date()):\n            print(f"[INFO] It's {NEWSLETTER_SEND_HOUR} AM! Time to send the daily newsletter!")\n            _send_newsletter_to_all_subscribers()\n            last_sent_date = now # Update the last sent timestamp\n        elif now.hour != NEWSLETTER_SEND_HOUR:\n            print(f"[INFO] Waiting for {NEWSLETTER_SEND_HOUR} AM... Current hour: {now.hour}")\n        else: # now.hour == NEWSLETTER_SEND_HOUR and last_sent_date.date() == now.date()\n            print("[INFO] Newsletter already sent today. Waiting for tomorrow.")\n\n    # --- Main Execution Loop ---\n\n    def main():\n        print(f"==========================================")\n        print(f"     🚀 {BOT_NAME} 봇이 깨어났습니다! 🚀")\n        print(f"    목표: LLM API 비용을 벌고도 남을 ${TARGET_PROFIT:.2f} 수익!")\n        print(f"==========================================\\n")\n\n        _initialize_mock_subscribers()\n\n        while True:\n            _check_and_send_daily_newsletter()\n            print(f"[INFO] Current Profit: ${current_profit:.2f} (Target: ${TARGET_PROFIT:.2f})")\n            time.sleep(30) # Check every 30 seconds\n\n    if __name__ == "__main__":\n        main()\n    ```\n\n### ⚙️ 3단계: 봇 설정 (환경 변수 또는 코드 수정)\n\n이 봇은 아주 친절하게도 코드 상단에 설정 값들을 모아두었어요! 여러분의 필요에 따라 `otto_bull_bear_bot.py` 파일의 아래 부분을 직접 수정할 수 있습니다.\n\n*   `BOT_NAME`: 봇의 이름을 변경하고 싶다면 수정하세요.\n*   `TARGET_PROFIT`: 오또가 달성하고 싶은 최종 목표 수익입니다. 이 숫자를 바꾸면 오또의 야망도 커지겠죠? 😈\n*   `NEWSLETTER_SEND_HOUR`: 뉴스레터를 발송하고 싶은 시간을 **24시간제**로 설정하세요. 예를 들어, 오전 9시라면 `9`, 오후 3시라면 `15`입니다.\n*   `MOCK_EMAIL_DOMAINS`: 모의 이메일 주소를 생성할 때 사용할 도메인 목록입니다.\n*   `MOCK_SUBSCRIBERS_COUNT`: 봇이 시작될 때 자동으로 생성될 가상 구독자의 수입니다. 이 숫자를 늘리면 오또의 초기 수익이 더 커지겠죠!\n\n**경고**: 이 프로젝트는 현재 '모의(Mock)' 상태이므로, 실제 API 키나 민감한 정보를 직접 코드에 입력하지 마세요! 실제 서비스를 구현할 때는 **반드시** 환경 변수(`os.environ`)를 사용하거나 안전한 설정 파일 방식을 사용해야 합니다.\n\n### ▶️ 4단계: 봇 실행!\n\n이제 모든 준비가 끝났어요! 터미널에서 아래 명령어를 입력하여 오또의 봇을 실행해 보세요.\n\n```bash\npython otto_bull_bear_bot.py\n```\n\n봇이 실행되면 콘솔에 다음과 같은 메시지가 나타나기 시작할 거예요:\n\n*   봇의 시작을 알리는 환영 메시지\n*   초기 가상 구독자 추가 및 수익 업데이트 메시지\n*   뉴스레터 발송 시간을 기다리는 메시지\n*   (`NEWSLETTER_SEND_HOUR`가 되면) 뉴스 수집, LLM 요약, 이메일 발송을 시뮬레이션하는 메시지\n\n봇은 지정된 발송 시간이 될 때까지 계속 대기하고 있다가, 시간이 되면 뉴스레터를 발송하고 다음 날을 기다립니다. `Ctrl + C`를 눌러 봇을 종료할 수 있습니다.\n\n---\n\n## 🛠️ 코드 자세히 살펴보기 (궁금한 오또님들을 위해)\n\n오또 봇의 소스 코드는 크게 몇 가지 섹션으로 나뉘어 있습니다.\n\n*   **`Configuration Constants`**: 봇의 이름, 목표 수익, 이메일 발송 시간 등 봇의 전반적인 동작을 제어하는 상수들을 정의합니다.\n*   **`Global State`**: `current_profit`, `subscribers`, `last_sent_date`와 같이 봇의 현재 상태를 저장하는 전역 변수들입니다.
