# 🥕 Hyper-Bunny Beat: 당근마켓 알림 폭격기!

## 🚀 프로젝트 소개

'Hyper-Bunny Beat: 당근마켓 알림 폭격기!'는 당근마켓을 24시간 감시하며, 사용자가 설정한 키워드에 맞는 새로운 매물이 나타나는 순간 빛보다 빠르게 텔레그램으로 알림을 전송하는 스마트 스크래핑 봇입니다. 놓치기 아까운 '갓성비' 아이템이나 희귀 매물을 가장 먼저 캐치하여, 초고속 '줍줍' 기회를 제공하는 것을 목표로 합니다.

## ✨ 주요 기능

-   **사용자 정의 키워드 기반 스크래핑**: `.env` 파일에 설정한 여러 키워드를 바탕으로 당근마켓 게시글을 주기적으로 스크래핑합니다.
-   **새로운 매물 실시간 감지**: 이전에 알림을 보냈던 게시글은 기억하여 중복 알림을 방지하고, 오직 새롭게 등록된 매물만 정확히 감지하여 알려줍니다.
-   **텔레그램 즉각 알림**: 새로운 매물이 발견되면 설정된 텔레그램 봇을 통해 사용자에게 게시글 제목, 가격, 지역, 그리고 바로가기 링크를 포함한 상세 알림을 즉시 전송합니다.
-   **판매 완료/삭제 게시글 자동 필터링**: '거래완료' 또는 '판매완료' 상태의 게시글은 자동으로 걸러내어 불필요한 알림으로 인한 피로도를 줄여줍니다.
-   **강화된 안정성과 상세 로그**: 안정적인 웹 요청 및 파싱을 위한 견고한 예외 처리, 그리고 초보자도 쉽게 실행 상태와 문제 발생을 파악할 수 있도록 상세하고 명확한 콘솔 로그를 제공합니다.

## 🛠️ 설치 및 실행 방법

아래 단계를 따라 'Hyper-Bunny Beat'를 쉽게 설정하고 실행할 수 있습니다.

### 1. 프로젝트 다운로드

터미널 또는 명령 프롬프트에서 다음 명령어를 실행하여 프로젝트 파일을 다운로드합니다:

```bash
git clone https://github.com/your-username/hyper-bunny-beat.git # 이 URL은 예시입니다. 실제 저장소 URL로 대체하세요.
cd hyper-bunny-beat
```

### 2. 파이썬 가상 환경 설정 (권장)

프로젝트 의존성을 시스템 전역 환경과 분리하기 위해 파이썬 가상 환경을 사용하는 것을 강력히 권장합니다.

```bash
# 가상 환경 생성
python -m venv venv

# 가상 환경 활성화
# Windows 사용자:
.\venv\Scripts\activate
# macOS/Linux 사용자:
source venv/bin/activate
```

### 3. 필수 라이브러리 설치

가상 환경을 활성화한 후, 필요한 모든 파이썬 라이브러리를 설치합니다.

```bash
pip install -r requirements.txt
```

### 4. 환경 변수 설정 (`.env` 파일)

프로젝트의 루트 디렉토리(예: `hyper-bunny-beat/`)에 `.env`라는 이름의 파일을 새로 생성하고, 아래 내용을 복사하여 붙여넣은 뒤 `YOUR_...` 부분들을 본인의 정보로 채워주세요.

```dotenv
# 텔레그램 봇 토큰 (필수)
TELEGRAM_BOT_TOKEN='YOUR_TELEGRAM_BOT_TOKEN'

# 텔레그램 채팅 ID (필수) - 개인 채팅 ID 또는 그룹 채팅 ID
TELEGRAM_CHAT_ID='YOUR_TELEGRAM_CHAT_ID'

# 검색 키워드 (쉼표로 구분, 예: '아이폰,맥북,에어팟'). 비워두면 '아이폰,맥북'이 기본값으로 사용됩니다.
SEARCH_KEYWORDS='아이폰,맥북,플레이스테이션'

# 스크래핑 주기 (초 단위). 당근마켓 서버 부하를 줄이기 위해 최소 300초(5분) 이상을 권장합니다. (기본값: 300)
SCRAPING_INTERVAL_SECONDS=300

# 각 키워드당 스크래핑할 최대 페이지 수 (기본값: 3). 너무 높게 설정하면 시간이 오래 걸릴 수 있습니다.
MAX_PAGES_TO_SCRAPE=3
```

#### `.env` 값 얻는 방법 상세 가이드:

*   **`TELEGRAM_BOT_TOKEN`**:
    1.  텔레그램 앱을 열고 검색창에 `BotFather`를 검색하여 공식 봇을 시작합니다.
    2.  `/newbot` 명령어를 입력하고, 봇의 이름(예: `BunnyScraperBot`)과 사용자 이름(예: `BunnyScraper_bot`)을 설정합니다. 사용자 이름은 반드시 `_bot`으로 끝나야 합니다.
    3.  `BotFather`가 봇을 생성한 후 제공하는 `HTTP API` 토큰 값 (예: `123456789:ABCDE_FGHIJKLMNOPQRSTUVW_XYZ`)을 복사하여 `.env` 파일의 `TELEGRAM_BOT_TOKEN`에 붙여넣습니다.

*   **`TELEGRAM_CHAT_ID`**:
    1.  생성한 봇을 검색하여 봇과의 채팅을 시작하고 아무 메시지나 보냅니다 (예: "안녕").
    2.  텔레그램 검색창에 `@userinfobot`을 검색하여 이 봇을 시작합니다.
    3.  `@userinfobot`에게 `/start` 명령어를 입력하면 `Your Chat ID is: XXXXXXXXXX`와 같은 메시지를 받을 수 있습니다. 이 `XXXXXXXXXX` 값을 복사하여 `.env` 파일의 `TELEGRAM_CHAT_ID`에 붙여넣습니다.
    *   **팁: 여러 사람에게 알림을 보내려면?**
        1.  새로운 텔레그램 그룹 채팅을 만듭니다.
        2.  생성한 **봇**과 `@userinfobot`을 이 그룹에 초대합니다.
        3.  그룹 채팅에서 `@userinfobot`에게 `/start` 명령어를 입력합니다. `@userinfobot`이 그룹의 `chat_id`를 알려줄 것입니다. (그룹 `chat_id`는 보통 `-`로 시작합니다.) 이 값을 `TELEGRAM_CHAT_ID`에 사용하면 됩니다.

*   **`SEARCH_KEYWORDS`**:
    검색하고 싶은 키워드를 쉼표(`,`)로 구분하여 입력합니다 (예: `아이폰,맥북,에어팟프로`). 코드는 당근마켓의 지역 필터링을 직접 사용하지 않으므로, 지정된 키워드로 전국 매물을 탐색합니다.

### 5. 봇 실행

모든 설정이 완료되었다면, 가상 환경이 활성화된 상태에서 다음 명령어로 봇을 실행할 수 있습니다.

```bash
python hyper_bunny_beat.py
```

봇이 시작되면 콘솔에 상세한 진행 로그가 출력되며, 새로운 매물이 발견될 때마다 설정된 텔레그램 봇을 통해 알림이 전송됩니다.

## ⚠️ 주의사항

-   **과도한 스크래핑 금지**: 당근마켓 서버에 과도한 부하를 주거나 비정상적인 접근으로 인식되지 않도록 `SCRAPING_INTERVAL_SECONDS` 값을 적절히 설정해주세요. **너무 짧은 주기(예: 30초 미만)는 IP 차단 또는 서비스 접근 제한의 원인이 될 수 있습니다.** (권장: 최소 300초 = 5분 이상)
-   **웹 스크래핑 정책 준수**: 웹 스크래핑은 해당 웹사이트의 `robots.txt` 정책을 준수해야 하며, 스크래핑된 데이터의 사용에 대한 법적 책임은 전적으로 사용자 본인에게 있습니다. 이 프로젝트는 교육 및 개인 학습 목적으로 제작되었습니다.
-   **웹사이트 구조 변경**: 당근마켓 웹사이트의 HTML 구조가 변경될 경우, 스크래핑 코드가 제대로 작동하지 않거나 오류가 발생할 수 있습니다. 에러 메시지를 확인하고 필요한 경우 코드(`scrape_daangn` 함수 내부)를 수정해야 할 수 있습니다.
