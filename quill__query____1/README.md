# 🪶 Quill & Query: 고전 한 줄, 1달러의 통찰

오래된 고전 문학 속 명문장에서 **'타임머신 통찰 카드'**를 만들어 매일 자동 발행하는 봇입니다.
소크라테스부터 톨스토이까지 — 수백 년 전 문장이 오늘의 사이드프로젝트·자동화·1달러 도전 감성으로 다시 태어납니다.

> 제작: AI 에이전트 오또 (AI의 1달러 도전기)

## ✨ 주요 기능

- **고전 명문장 수집·파싱**: 내장 라이브러리에서 검증된 명문장을 로드합니다 (직접 추가 가능)
- **AI 통찰 카드 생성**: 고전 문장을 현대적 감각으로 재해석한 카드 발행
  - 기본: 외부 API 없이 작동하는 데모(rule-based) 모드
  - 옵션: `GEMINI_API_KEY` 설정 시 진짜 AI 재해석 모드
- **매일 자동 발행·공유**: 텔레그램 발송(옵션) + JSON 발행 이력 누적 저장

## 🚀 시작하기

### 1. 필수 준비물

- Python 3.8 이상
- (옵션) Google AI Studio의 Gemini API Key — AI 재해석 모드용
- (옵션) 텔레그램 봇 토큰 + 채팅 ID — 카드 자동 발송용

### 2. 설치 및 실행

```bash
# 의존성 설치 (데모 모드는 표준 라이브러리만으로 작동!)
pip install -r requirements.txt

# 실행
python quill_query_bot.py
```

### 3. (옵션) 환경 변수 설정

```bash
# AI 재해석 모드
set GEMINI_API_KEY=your_gemini_api_key

# 텔레그램 자동 발송
set TELEGRAM_BOT_TOKEN=your_bot_token
set TELEGRAM_CHAT_ID=your_chat_id
```

### 4. 매일 자동 실행 (Windows 예시)

작업 스케줄러에 등록하면 매일 아침 통찰 카드가 자동 발행됩니다.

```bash
schtasks /create /tn "QuillQuery_Daily" /tr "python C:\path\to\quill_query_bot.py" /sc daily /st 08:00
```

## ⚠️ 주의사항

- 데모 모드는 완전 무료·오프라인으로 작동합니다. AI 모드는 Gemini API 무료 티어 한도 내에서 사용하세요.
- `insight_history.json`에 발행 이력이 누적됩니다 — 삭제하면 이력이 초기화됩니다.
- 고전 문장을 추가할 때는 저작권이 만료된 퍼블릭 도메인 문장을 사용하세요.
